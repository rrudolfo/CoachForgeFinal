Design a 6-page responsive website for a project called Coach Forge. The site should feel like a systemic culture operating model presented through an editorial, high-performance visual language. It should combine the clarity of a performance system with the energy of a sports strategy board, archival research artifact, and poster-based design world.

Core project framing

This website presents a culture and performance operating system for an AI + Design project, built around this core model:

Culture Mechanics × Coach Practices × AI Functions

The system shows how:

standards, safety, and collective efficacy reinforce performance

AI supports capture, pattern detection, and nudges

humans remain the authority layer

coach routines become repeatable organizational rituals

This is a portfolio-quality systems site, not a plain information site.
